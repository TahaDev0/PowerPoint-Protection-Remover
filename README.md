### 🚀 طريقة الاستخدام

1. **التحضير**: احفظ نسخة احتياطية من ملفاتك المهمة.
2. **التشغيل**: افتح التطبيق (اختر العربية أو الإنجليزية).
3. **الموافقة**: اقرأ إخلاء المسؤولية واضغط على "موافق".
4. **إضافة الملفات**:
   - اسحب وأفلت الملفات في المنطقة المخصصة.
   - أو اضغط على زر "إضافة ملفات PowerPoint".
5. **الانتظار**: سيعالج التطبيق الملفات تلقائياً في الخلفية.

#### ⚠️ تنبيه مهم بعد إزالة حماية VBA

1. افتح الملف المعالج في PowerPoint.
2. ادخل إلى محرر VBA (Alt + F11).
3. قد تظهر رسالة خطأ "Invalid Key" أو ما شابه - اضغط OK.
4. احفظ الملف (Ctrl + S) ثم أغلقه.
5. أعد فتح الملف - ستجد محرر VBA مفتوحاً بدون كلمة مرور.

<div id="removed-protections"></div>

### 🔒 أنواع الحماية التي تتم إزالتها

*   **حماية العرض التقديمي (Presentation Protection)**: تزيل كلمة مرور التعديل وتفتح العرض للتحرير.
*   **حماية VBA (VBA Project Protection)**: تزيل كلمة مرور مشروع VBA لتمكين الوصول إلى الأكواد.
*   **وضع القراءة فقط (Read-Only Recommended)**: تزيل التوصية بفتح العرض التقديمي كقراءة فقط.
*   **حماية الكتابة (Write Protection)**: تسمح بحفظ التغييرات على الملف دون قيود.
*   **حماية الشرائح الفردية**: تزيل القيود التي قد تكون مطبقة على شرائح معينة.

---

<div id="processing-methods"></div>

### ⚙️ طرق المعالجة المختلفة

*   `process_zip_format()`: تستخدم لمعالجة ملفات PowerPoint الحديثة (pptx, pptm, potx, potm, ppsx, ppsm) التي تعتمد على بنية Zip.
*   `process_ole_format()`: تستخدم لمعالجة ملفات PowerPoint القديمة (.ppt) التي تعتمد على بنية OLE.

---

<div id="important-notes"></div>

### ⚠️ ملاحظات مهمة

*   **كلمة مرور الفتح (Open Password)**: هذا البرنامج **لا يستطيع إزالة أو استعادة كلمة مرور الفتح** للعرض التقديمي. هو يزيل فقط حماية التعديل والقيود الأخرى.
*   **الملفات القديمة (.ppt)**: قد لا تكون إزالة الحماية فعالة بنسبة 100% مع جميع أنواع الحماية في ملفات .ppt القديمة بسبب اختلاف بنيتها.
*   **النسخ الاحتياطي**: مهم جداً أخذ نسخة احتياطية من ملفاتك قبل البدء بعملية المعالجة.
*   **التوافق**: إصدارات PowerPoint المختلفة قد تتعامل مع الحماية وطرق إزالتها بطرق مختلفة، مما قد يؤثر على فعالية الأداة في بعض الحالات.

### 🚀 How to Use

1. **Preparation**: Backup your important files first.
2. **Launch**: Open the application (choose Arabic or English).
3. **Accept**: Read the disclaimer and check "I agree".
4. **Add files**:
   - Drag and drop files into the drop area.
   - Or click "Add PowerPoint files" button.

   - Click any file to save it individually.

#### ⚠️ Important Note After VBA Protection Removal

2. Enter VBA editor (Alt + F11).

5. Reopen the file - VBA editor will open without password.

<div id="Eremoved-protections"></div>

### 🔒 Types of Protections Removed

*   **Presentation Protection**: Removes the modification password and opens the presentation for editing.
*   **VBA Protection**: Removes the VBA project password to enable access to code.
*   **Read-Only Recommended**: Removes the recommendation to open the presentation as read-only.
*   **Write Protection**: Allows saving changes to the file without restrictions.
*   **Individual Slide Protection**: Removes restrictions that may be applied to specific slides.

---

<div id="Eprocessing-methods"></div>

### ⚙️ Different Processing Methods

*   `process_zip_format()`: Used for modern PowerPoint files (pptx, pptm, potx, potm, ppsx, ppsm) based on Zip structure.
*   `process_ole_format()`: Used for legacy PowerPoint files (.ppt) based on OLE structure.

---

<div id="Eimportant-notes"></div>

### ⚠️ Important Notes

*   **Open Password**: This software **cannot remove or recover the open password** for a presentation. It only removes modification protection and other restrictions.
*   **Legacy Files (.ppt)**: Protection removal might not be 100% effective with all types of protection in older .ppt files due to their different structure.
*   **Backup**: It is crucial to create a backup of your files before starting the processing.
*   **Compatibility**: Different PowerPoint versions may handle protection and removal methods differently, which might affect the tool's effectiveness in some cases.
